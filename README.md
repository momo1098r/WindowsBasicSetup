# WindowsBasicSetup
Basic setup after Install windows

A two-file automation script (setup.cmd + setup.ps1) for quick Windows configuration after a fresh install. Applies registry tweaks, installs software via Chocolatey, downloads Office 2024, and runs Windows Update in a loop until the system is fully up to date — rebooting and resuming automatically via Scheduled Task.
Features:

Dark theme, classic context menu, file extensions visible
Custom wallpaper & lock screen
UAC disabled, power plan optimized, boot timeout set to 3s
Chocolatey + Chrome, Firefox, VLC, 7-Zip, Everything, TeraCopy, Adobe Reader, HWiNFO, Java, K-Lite Codec Pack
Office 2024 installer (direct from Microsoft)
Automatic Windows Update loop with reboot handling
